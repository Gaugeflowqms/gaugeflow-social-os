## Example daily post hooks

- "The audit problem is usually not the auditor."
- "Good parts still need clean records."
- "Traceability fails quietly until someone asks for proof."
- "Spreadsheets work until three people edit three different versions."
- "A cert packet should not be a scavenger hunt."
- "FAI should not depend on someone remembering where the file went."
- "Revision control is not exciting. Missing a revision is worse."

- "Inspection data should follow the job, not live in someone's desktop folder."
- "Calibration records matter most when nobody has time to look for them."
